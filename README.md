<h1 align="center">Gestão de cinemas</h1>
<p>>Aplicação de gestão de cinemas utilizando uma arquitetura orientada a micro serviços.
A aplicação permitia efetuar reservas, fazer gestão dos lugares, comprar bebidas , fazer pagamentos entre outros.</p>
